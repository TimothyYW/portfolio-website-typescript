# Timothy Y Wewengkang — Portfolio

A dark techy personal portfolio website built with **Next.js 16**, **React 19**, and **Tailwind CSS v4**. Styled with a red, black, and white colour scheme.

## About

This is the personal portfolio of Timothy Y Wewengkang — a web development student and Master of IT (Software Development) candidate at Central Queensland University, Brisbane.

## Sections

- **Hero** — Introduction and call to action
- **About Me** — Background, focus, and current status
- **Skills & Technologies** — Frontend, backend, and DevOps tools
- **Portfolio** — Showcased projects with STAR methodology breakdowns
- **Experience** — Education and professional journey
- **Testimonials** — Client and colleague feedback
- **Contact** — Direct contact details and social links

## Tech Stack

| Category | Technology |
|---|---|
| Framework | Next.js 16 |
| UI Library | React 19 |
| Language | TypeScript |
| Styling | Tailwind CSS v4 |
| Icons | react-icons |
| Fonts | Geist Sans & Geist Mono |

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
